# 🎬 Video Rental System

Sistema de alquiler de materiales audiovisuales desarrollado en Java como parte del curso de Programación Orientada a Objetos.

## 📌 Descripción

El sistema permite gestionar usuarios, un catálogo de materiales (VHS, DVD y BluRay) y las operaciones básicas de alquiler y devolución.  
Fue diseñado aplicando principios fundamentales de la Programación Orientada a Objetos como abstracción, encapsulamiento, herencia y polimorfismo.

## ⚙️ Funcionalidades

- Registro de usuarios (clientes y empleados)
- Gestión de catálogo de materiales
- Alquiler de materiales
- Devolución de materiales
- Consulta de préstamos activos

## 🧱 Estructura

El proyecto se organiza en las siguientes capas:

- `model`: clases del dominio (Usuario, Material, etc.)
- `system`: lógica central del sistema
- `ui`: interacción por consola (menú)

## 📚 Contexto

El sistema modela un Video Rental tradicional, un tipo de negocio hoy en día obsoleto debido al streaming, pero que permite representar claramente entidades y relaciones para efectos de aprendizaje en POO.
